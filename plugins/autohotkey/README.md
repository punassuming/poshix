# autohotkey plugin

Helpers for launching and editing AutoHotkey scripts.

## Commands

- `Start-AutoHotkeyScript <path> [args...]` (alias: `ahk`)
- `Edit-AutoHotkeyScript <path>` (alias: `ahk-edit`)

## Notes

- AutoHotkey must be installed and available in `PATH` (`AutoHotkey64.exe`, `AutoHotkey.exe`, or `autohotkey`).
