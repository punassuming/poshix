# tortoisegit plugin

Helper command for opening TortoiseGit dialogs.

## Commands

- `Invoke-TortoiseGit [command] [path]` (alias: `tgit`)

Examples:

- `tgit` (opens log dialog for current directory)
- `tgit commit .`
- `tgit push C:\src\my-repo`

## Notes

- Requires TortoiseGit (`TortoiseGitProc.exe`).
